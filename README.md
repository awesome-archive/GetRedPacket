# GetRedPacket

微信自动抢红包 ，目前支持版本7.0

核心还是 AccessibilityService 监听 参考了其他的库（间隔时间比较久忘记github中是哪个）修改了参考项目的bug


使用方法： 安装成功后 点击“开启/关闭 抢红包”-跳转至无障碍设置页面 打开---“红包demo” 切换至手机主页，确保 微信 7.0 以及 微信 和 抢红包 存在后台
